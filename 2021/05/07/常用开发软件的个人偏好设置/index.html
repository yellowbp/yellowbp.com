常用开发软件的个人喜好配置

下面是我个人常用，看得舒服的配置，有idea、maven

IDEA

导入Eclipse快捷键

因为个人使用IDE的情况不同，对于从Eclipse到IDEA的老玩家来说，IDEA自带可以导入其他IDE的快捷键可谓相当友好

  1. 依次点击菜单栏中的“File”->“Import Settings”
  2. 选择需要的快捷键jar包(需要的可以邮箱留言)
    • 那么为什么不在settings –> Keymap –> 选择Eclipse呢
    • 因为那里的Eclipse的快捷键不全，所以我们需要导入一个全的jar包
  3. 一直点ok，然后重启即可
  4. 检验是否导入成功：依次点击菜单栏中的 File –> Settings –> Keymap 中我们就可以看到导入的jar包了

以下操作均在 File –> settings中进行

设置鼠标悬浮提示

Editor --> General --> 勾选show quick documentation on mouse move

字体

  1. 设置鼠标滚轮滚动修改字体大小：

    Editor --> General --> 勾选change font size(Zoom) with Ctrl+Mouse Wheel

  2. 敲代码时自动导入开发包：

    Editor --> General --> 勾选Add unambiguous imports on the fly 和 Optimize imports on the fly

  • Add unambiguous imports on the fly：自动导入不明确的结构
  • Optimize imports on the fly：自动帮我们优化导入的包
  1. 编辑区字体大小

Editor --> General --> Font

我喜欢的字体：consolas 大小：18 行距：1.2

  1. 控制台输出的字体

Editor --> Color Scheme--> Console Font --> 勾选Use console font instread of the default --> 再选择自己喜欢的字体等

设置多行显示标签页(Tabs)的操作

Editor --> General --> Editor Tabs --> 取消勾选show tabs in one row

设置项目文件编码

Editor --> File Encodings--> 都用utf8且勾选Transparent native-to-ascii conversion

  • Transparent native-to-ascii conversion：主要用于转换 ascii，一般都要勾选， 不然 Properties 文件中的注释显示的都不会是中文。
